Fuel System Isolation Procedure
ISM Safety Procedure
ISM-F042 · Rev. 3 · Effective 01 Jan 2026
SV Andromeda · 60m · Engineering Department
Safety Critical — Read Before Commencing Work
1. Scope & Application

This procedure applies to all fuel system isolation operations on main engine and generator circuits aboard SV Andromeda. It covers emergency and planned isolation scenarios for both port and starboard fuel trains.

2. Pre-Isolation Safety Checks

Before commencing isolation, confirm: (a) all running machinery on affected circuit has been shut down, (b) bridge has been notified and acknowledges reduced propulsion capability, (c) fire watch is stationed at isolation point, (d) portable extinguisher is within 3m of work area.

3. Isolation Valve Sequence

Close valves in the following order: V-101 main supply, V-102 return line, V-103 day tank feed. Wait 60 seconds between each closure to allow pressure equalisation. Verify zero pressure on gauge PG-F04 before proceeding to bypass installation.

4. Bypass Installation

Install temporary bypass spool piece between flanges F-12A and F-12B. Torque bolts to 45 Nm in star pattern. Conduct 30-minute pressure test at 4 bar before returning system to service. Record test result in engine room log.
