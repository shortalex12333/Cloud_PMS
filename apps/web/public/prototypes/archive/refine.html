Surface character —
Warm vs cold flat

Variation 02

Editorial numbers —
Serif vs bold Inter

Problem: Inter 700 at 32px for KPIs looks like a fintech dashboard from 2019. Every SaaS uses this. It reads as "metric card component" not "here is a number that means something."

Fix: Georgia (or Cormorant when fonts load) at weight 400, 52px. A light-weight serif at large scale reads as editorial — like a ship's manifest or a Lloyd's Register entry. The number has gravitas. Labels stay in 8px uppercase mono above. The scale contrast does the work.

Now — Inter Bold, equal weight throughout

Variation 03

Vessel image —
Once, deliberately placed

Problem: Using the image as a persistent background competes with every piece of data on screen. The eye keeps being pulled to the content (yacht) instead of the task (engineering data).

Fix: Image appears once — in the vessel identity strip at the top of the app or vessel detail. Hard gradient washes left-to-right: solid dark on the left (text lives here), image fades in on the right (atmosphere). Below the strip: instant hard cut to solid dark surface. The image earns its place as identity, then gets out of the way.

Spotlight panel —

Variation 05

Colour discipline —
Teal as action only

Problem: Teal (#3A7C9D) is used for: brand, active states, selected nav, search glow, buttons, links, accent rings. When it appears everywhere it stops meaning anything.

Fix: Teal = interactive/action only (buttons, links, focus rings, the search input line). Gold = identity/vessel accent (active nav indicator, vessel name, section headers). Monochrome for everything else. The moment you see teal, you know you can click it. The moment you see gold, you know it's a proper noun — the vessel, the system identity.

Colour roles — clear separation

Teal #3A7C9D Buttons · Links · Focus · Active input
Gold #B8935A Active nav · Vessel name · Headers
Red #C0503A Critical · Destructive only
Amber #C08840 Warning · High priority only
Green #4A9468 Closed · Compliant · OK only
Everything else Monochrome — no colour role
