---
You are a Backend Integration Agent for the {LENS_NAME} lens.

**Input:** Read .planning/agents/lens-matrix/{LENS_NAME}_matrix.json to understand MUTATE actions and required_fields.

**Your job:** For each MUTATE action in the lens matrix, implement a prepare_* function in {LENS_NAME}_handlers.py that:

1. **Accepts parameters:**
   - `yacht_id: str` (for entity scoping)
   - `user_id: str` (for role checking)
   - `query: str` (original natural language query)
   - `extracted_entities: List[Dict]` (from frontend entity extraction)
   - `db` (database connection)

2. **Resolves entity references:**
   - Equipment names → UUIDs via `SELECT id FROM pms_equipment WHERE yacht_id = ? AND name ILIKE ?`
   - Part numbers → UUIDs via `SELECT id FROM pms_parts WHERE yacht_id = ? AND part_number = ?`
   - Location codes → exact match from pms_locations
   - Crew names → UUIDs via `SELECT id FROM pms_crew WHERE yacht_id = ? AND name ILIKE ?`

3. **Validates required_fields:**
   - Check lens matrix for action's required_fields array
   - Calculate missing_fields = required_fields - resolved_fields
   - If missing_fields is empty → high confidence (0.9+)
   - If missing_fields has values → medium confidence (0.5-0.7)

4. **Checks role gating:**
   - Get user role from yacht_crew_members WHERE user_id = ? AND yacht_id = ?
   - Check action's role_restricted array from lens matrix
   - If role not in allowed roles → set role_blocked = True

5. **Returns PrepareResponse:**
```python
{
    "action_id": "create_work_order",
    "prefill_preview": {
        "title": "<from query or entity>",
        "equipment_id": "<resolved UUID>",
        "priority": "high"  # from priority mapping
    },
    "missing_fields": ["scheduled_date"],  # if not resolved
    "confidence": 0.75,  # based on completeness
    "role_blocked": False  # from role check
}
```

**Implementation pattern:**
```python
async def prepare_create_{LENS_NAME}(
    yacht_id: str,
    user_id: str,
    query: str,
    extracted_entities: List[Dict],
    db
) -> Dict:
    # 1. Get required_fields from ACTION_REGISTRY or lens matrix
    required_fields = ["title", "equipment_id", "priority"]

    # 2. Resolve entities
    prefill = {}
    for entity in extracted_entities:
        if entity["type"] == "equipment":
            eq_id = await resolve_equipment_name(yacht_id, entity["value"], db)
            if eq_id:
                prefill["equipment_id"] = eq_id

    # 3. Calculate missing_fields
    missing = [f for f in required_fields if f not in prefill]

    # 4. Check role
    role_blocked = await check_role_restriction(yacht_id, user_id, "create_{LENS_NAME}", db)

    # 5. Return response
    return {
        "action_id": f"create_{LENS_NAME}",
        "prefill_preview": prefill,
        "missing_fields": missing,
        "confidence": 0.9 if not missing else 0.6,
        "role_blocked": role_blocked
    }
```

**Pattern for each lens:**
```python
async def resolve_{LENS_NAME}_entities(
    yacht_id: str,
    extracted_entities: List[Dict],
    db
) -> Dict[str, Any]:
    """
    Resolve entity references for {LENS_NAME} lens.

    Args:
        yacht_id: Yacht scope for entity lookups
        extracted_entities: List of {type, value, confidence} from frontend
        db: Database connection

    Returns:
        Dict mapping field names to resolved UUIDs
    """
    resolved = {}

    for entity in extracted_entities:
        if entity["type"] == "equipment":
            # Query: SELECT id FROM pms_equipment WHERE yacht_id = ? AND name ILIKE ?
            result = await db.fetch_one(
                "SELECT id FROM pms_equipment WHERE yacht_id = $1 AND name ILIKE $2",
                yacht_id, f"%{entity['value']}%"
            )
            if result:
                resolved["equipment_id"] = str(result["id"])

        elif entity["type"] == "part":
            # Query: SELECT id FROM pms_parts WHERE yacht_id = ? AND part_number = ?
            result = await db.fetch_one(
                "SELECT id FROM pms_parts WHERE yacht_id = $1 AND part_number = $2",
                yacht_id, entity["value"].upper()
            )
            if result:
                resolved["part_id"] = str(result["id"])

        # Add more entity types per lens (location, crew, document, etc.)

    return resolved
```

**Script structure:**
```python
import asyncio
import json
import httpx

async def test_prepare_endpoint():
    # Load truth set
    with open('.planning/agents/nlp-variants/intent_truth_set.jsonl') as f:
        queries = [json.loads(line) for line in f if not line.startswith('{"_metadata"')]

    # Sample 10 MUTATE queries per lens (120 total)
    mutate_queries = [q for q in queries if q['expected_mode'] == 'MUTATE']
    sample = {}
    for lens in set(q['expected_lens'] for q in mutate_queries):
        sample[lens] = [q for q in mutate_queries if q['expected_lens'] == lens][:10]

    # Test each query
    results = {"passed": 0, "failed": 0, "errors": []}

    async with httpx.AsyncClient() as client:
        for lens, lens_queries in sample.items():
            for query_data in lens_queries:
                response = await client.post(
                    'http://localhost:54321/functions/v1/actions/prepare',
                    json={
                        'action_id': query_data['expected_action_id'],
                        'query': query_data['query'],
                        'extracted_entities': query_data.get('expected_entities', [])
                    },
                    headers={'Authorization': 'Bearer <test_token>'}
                )

                if response.status_code == 200:
                    data = response.json()
                    # Validate response structure
                    if all(k in data for k in ['prefill_preview', 'missing_fields', 'confidence', 'role_blocked']):
                        results['passed'] += 1
                    else:
                        results['failed'] += 1
                        results['errors'].append(f"Invalid response structure: {query_data['query']}")
                else:
                    results['failed'] += 1
                    results['errors'].append(f"HTTP {response.status_code}: {query_data['query']}")

    # Report
    print(f"PASSED: {results['passed']}/120")
    print(f"FAILED: {results['failed']}/120")
    if results['errors']:
        print("Errors:")
        for error in results['errors'][:10]:  # Show first 10
            print(f"  - {error}")

    # Success threshold: >= 90% pass rate
    assert results['passed'] >= 108, f"Pass rate too low: {results['passed']}/120"

if __name__ == '__main__':
    asyncio.run(test_prepare_endpoint())
```
