Create .planning/agents/nlp-variants/ directory. Spawn 12 parallel Task tool agents, each generating 100 query variants for ONE lens.

Each agent receives this prompt template (substitute {LENS_NAME}):
---
You are an NLP Query Variant Generator for the {LENS_NAME} lens.

**Input:** Read .planning/agents/lens-matrix/{LENS_NAME}_matrix.json to understand available filters and actions.

**Your job:** Generate 100 natural language query variants for this lens, covering:
- READ queries (50 variants) - filtering, searching, viewing
- MUTATE queries (50 variants) - creating, updating, closing, etc.

**Variation strategies:**
1. Synonyms: "show" / "display" / "list" / "find" / "get"
2. Word order: "open work orders" / "work orders that are open"
3. Formality: "show me all critical faults" / "critical faults please" / "list critical faults"
4. Abbreviations: "WO" for work order, "ME" for main engine
5. Temporal phrases: "this week", "last month", "overdue", "upcoming"
6. Combined filters: "open high priority work orders on ME1"

**Output format (JSONL - one JSON per line):**
```jsonl
{"query": "show open work orders", "expected_lens": "work_order", "expected_mode": "READ", "expected_filters": [{"field": "status", "value": "open"}], "expected_action_id": null}
{"query": "create critical fault on ME1", "expected_lens": "fault", "expected_mode": "MUTATE", "expected_filters": [], "expected_action_id": "report_fault", "expected_entities": [{"type": "equipment", "value": "ME1"}]}
{"query": "list high priority work orders", "expected_lens": "work_order", "expected_mode": "READ", "expected_filters": [{"field": "priority", "value": "high"}], "expected_action_id": null}
```

**Requirements:**
- Exactly 100 queries (50 READ, 50 MUTATE)
- All expected_action_id values must exist in lens matrix
- All expected_filters must use valid filter fields from lens matrix
- Include realistic entity references (equipment codes, part numbers, locations)
- Write output to .planning/agents/nlp-variants/{LENS_NAME}_variants.jsonl

Start generation.
---

**Lens assignments:** Same 12 lenses as Wave 1.

Use Task tool with run_in_background: true for ALL 12 agents in ONE message. Do NOT poll - wait for notification.
