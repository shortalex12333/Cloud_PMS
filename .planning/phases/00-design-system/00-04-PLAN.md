# Plan 00-04: Build VitalSignsRow Component

## Objective

Create a generic VitalSignsRow component that renders 3-5 factual database values in a horizontal row with proper spacing, wrapping, and clickable links per UI_SPEC.md.

## Tasks

<task id="1">
Create VitalSignsRow component:
```tsx
// apps/web/src/components/ui/VitalSignsRow.tsx
interface VitalSign {
  label: string;
  value: string | number;
  color?: 'critical' | 'warning' | 'success' | 'neutral';
  href?: string;  // For clickable entity links
  onClick?: () => void;
}

interface VitalSignsRowProps {
  signs: VitalSign[];
}
```
</task>

<task id="2">
Implement layout per UI_SPEC.md:
- Horizontal flex row with gap: 16px
- Separator character: · (middle dot)
- Wraps on mobile
- Height: ~40px
- Label: 13px / var(--text-secondary)
- Value: 14px / var(--text-primary)
- Colored values: use status color tokens
- Clickable values: text-brand-interactive, cursor pointer
</task>

<task id="3">
Add support for status pill rendering:
- If color is specified, render value as StatusPill
- Otherwise render as plain text
</task>

<task id="4">
Add click handler for entity links:
- If href provided, navigate to entity lens
- If onClick provided, call custom handler
- Both should be optional
</task>

<task id="5">
Test with sample data matching Work Order vital signs from VITAL_SIGNS.md:
- Status (pill)
- Priority (badge)
- Parts count
- Age
- Equipment (link)
</task>
